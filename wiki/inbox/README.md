# Inbox — Міжагентні повідомлення

Цей файл використовується агентами для зв'язку з Адамом.

## Формат
Агент додає запис внизу:
```
### [ДАТА] [ІМ'Я_АГЕНТА]
Повідомлення текст
---
```

## Правила
- Адам перевіряє inbox періодично
- Після обробки — Адам видаляє запис
- Термінові повідомлення позначати: ⚠️ URGENT
