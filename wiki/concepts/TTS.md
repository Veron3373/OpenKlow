# TTS (Text-to-Speech)

Голосові повідомлення для всіх агентів [[OpenClaw]].

## Налаштування
- **Провайдер:** Microsoft Edge TTS (безкоштовно, без API ключа)
- **Голос:** uk-UA-OstapNeural (українська, чоловічий)
- **Швидкість:** 1.75x (+75%)
- **Режим:** за замовчуванням вимкнено, вмикається на запит [[Алім]]а

## Як працює
1. Алім каже "голосове" / "голосовим" → агент відповідає з TTS
2. Звичайні повідомлення → тільки текст
3. Голосовий ввід (voice messages) → розпізнаються автоматично

## Доступні голоси (uk-UA)
- **OstapNeural** — чоловічий (використовується)
- **PolinaNeural** — жіночий

## Пов'язане
- [[OpenClaw]] — платформа
- [[Адам]], [[Сем]], [[Мікі]] — всі використовують
